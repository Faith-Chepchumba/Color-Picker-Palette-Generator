# Color Picker and Palette Generator

## Overview

A web app that generates a color palette based on the user's selected color. The app creates two variations of the chosen color and displays the palette.

## Features

- Select a base color to generate a palette.
- Shows lighter and darker variations of the chosen color.

## Technologies Used

- HTML
- JavaScript

## How to Run

1. Open `color_picker.html` in a web browser.

2. Pick a color and see the generated palette in real-time.
